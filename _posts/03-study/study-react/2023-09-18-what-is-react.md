# SPA (Single Page Application)

---

리액트의 특징중 하나로 Single Page Application 이라는 점이 있다.   

Multi Page Application 인 MPA 와 비교하자면  
리렌더더링시 페이지 갱신이 일어나 화면이 깜빡거리게 되지만  

SPA 는 서버에 1회만 리소스 요청을 하고  
그 후에는 필요할 때  
데이터만 받아와 기존페이지를 수정해주는 방식으로 작동하게 된다.
